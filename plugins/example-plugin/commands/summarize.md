# Summarize Command

Analyze and summarize the content provided by the user.

## Behavior

1. If $ARGUMENTS contains text, summarize that text
2. If no arguments are provided, summarize the currently open file
3. Provide a concise summary with key points

## Output Format

- Brief overview (1-2 sentences)
- Key points as bullet list
- Any important notes or considerations
